My dev setup

In this post I list all the programs and configurations I use for my development workflow. I favor a terminal centric approach in the tools I use as I consider it to be the most efficient way to work.

Tools

  • Code editor: Neovim with vim-plug
  • Shell: fish with oh-my-fish and spacefish
  • Terminal emulator: kitty
  • tig

Optional

  • Command line finder: fzf
  • Command line color scheme: Gogh
  • Vim browser extension: Vimium

Alternatives

  • Tmux
  • Github-desktop
  • Vscode
  • i3
  • zsh with oh-my-zsh and spaceship theme
